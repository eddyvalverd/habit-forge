# Acceptance Criteria

The app is done only if:

1. User can create a habit.
2. Empty habit names are rejected.
3. User can mark a habit as completed today.
4. User can unmark a habit for today.
5. User can delete a habit.
6. Habits persist after page refresh.
7. Current streak is visible.
8. App works on mobile width.
9. No TypeScript errors.
10. Build passes.
11. Lint passes.
12. README explains how to run the app.
