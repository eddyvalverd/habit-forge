# Habit Forge - PRD

## Goal

Build a simple habit tracking app.

## User

A person who wants to track daily habits without creating an account.

## MVP Features

1. Create a habit with a name.
2. Show all habits in a list.
3. Mark/unmark a habit as completed today.
4. Show current streak for each habit.
5. Delete a habit.
6. Persist data in localStorage.
7. Responsive design for mobile and desktop.

## Out of Scope

- Authentication
- Database
- Backend
- Payments
- Social features
- Notifications
- Calendar view
- AI features

## Main Screen

The app should have one dashboard page with:

- app title
- habit creation input
- add button
- habit cards
- completed today indicator
- streak number
- delete button

## Success

The user can open the app, create habits, complete them, refresh the browser, and still see their data.
